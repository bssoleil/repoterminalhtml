Les bases

  • Ouvrir le terminal : Ctrl Alt T
  • Fermer le terminal : exit
  • Nettoyer le terminal : clear
  • Répète la dernière commande : !!
  • Annuler/Abondonner : Ctrl C
  • Auto-complétion : TAB
  • Installer : sudo apt-get install

Les permissions

  • Ordre : user/group/all
    r : lire
    w : écrire/modifier
    x : executer
  • modifier les permissions : chmod

Types de dossiers/Déplacements

  • Montre le répertoire actuel : .
  • Répertoire parent : ..
  • Home : ~
  • Changer de répertoire : cd (nom)
  • Retourner dans le dossier précédent : cd -
  • Retourner sur HOME : cd ou cd ~
  • Savoir son emplacement : pwd

Création/Dossiers/Fichiers

  • Créer un dossier : mkdir:
  • créer un fichier : touch nom.format

Renommer/Supprimer

  • Renommer un fichier : mv (ancien nom) (nouveau nom)
  • Supprimer un fichier :rm nom du fichier
  • Supprimer un dossier vide : rmdir nom du fichier
  • Supprimer un dossier et son contenu : rm -r nom

Affichage Dossiers/Fichiers

  • Contenu du dossier : ls
  • Contenu du dossier + fichiers cachés : ls -a
  • Contenu du dossier + permissions : ls -l
  • \\ dossier + fichiers cachés + permissions : ls -la
